The sample dataset includes representative image pairs from multiple disaster types:
- 🌋 Volcano
- 🌎 Earthquake
- 🔥 Wildfire

Each sample consists of Pre & Post disaster image (.png) as well as Building annotation JSON files for both timestamps.

Note: The complete xView2 dataset contains over 10,000 multi-temporal satellite image pairs (in .tif format) and is not included in this repository due to its large size.
