The sample dataset includes representative image pairs from multiple disaster types:
- Volcano 🌋
- Earthquake 🌎 
- Wildfire 🔥
  
Each sample consists of Pre- and Post-disaster images along with the corresponding building annotation JSON files for both timestamps.

The original xView2 dataset is organized into four splits:
- **Tier 1 + Tier 3** → Training set
- **Hold** → Validation set
- **Test** → Test set

> **Note:** The complete xView2 dataset contains over 10,000 multi-temporal satellite image pairs (in `.tif` format) and is **not included** in this repository due to its large size.
